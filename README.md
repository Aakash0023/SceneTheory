# 🎬 SceneTheory

**SceneTheory** is a modern AI-powered movie discovery platform built for movie enthusiasts. It combines movie exploration, AI-generated quizzes, community discussions, and personalized watchlists into a single cinematic experience. Users can discover trending movies, explore detailed information, watch trailers, test their knowledge with AI-generated quizzes, and share their thoughts with fellow movie lovers.

## ✨ Features

- 🎥 Discover trending and popular movies
- 🔍 Real-time movie search
- 📖 Detailed movie information and trailers
- 🧠 AI-powered movie quizzes
- ❤️ Personal watchlist management
- 👥 Community page for sharing movie reviews
- 👤 Personalized user profile with movie statistics
- 📱 Fully responsive and modern cinematic UI

## 🛠️ Tech Stack

- **Frontend:** React, Vite, JavaScript
- **Styling:** CSS3, Framer Motion, React Icons
- **APIs:** TMDB API, Groq AI
- **Deployment:** Vercel
